 Worker Income System (Java)


Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO), que calcula a renda de um trabalhador com base em seu salário base e contratos por hora em um determinado período.

Neste projeto eu pratiquei conceitos como:

- Classes e objetos
- Encapsulamento
- Composição entre classes
- Enumerações
- Manipulação de datas
- Organização de pacotes
